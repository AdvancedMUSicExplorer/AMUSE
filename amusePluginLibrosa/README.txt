For Librosa:
------------------
To use Librosa within AMUSE, please install it manually as Python module using, 
e.g., pip.

LibROSA is a python package for music and audio analysis. It provides the 
building blocks necessary to create music information retrieval systems.

For a quick introduction to using librosa, please refer to the Tutorial
<https://librosa.github.io/librosa/tutorial.html>. 
For a more advanced introduction which describes the package design principles, 
please refer to the librosa paper 
<http://conference.scipy.org/proceedings/scipy2015/pdfs/brian_mcfee.pdf> 
at SciPy 2015.
